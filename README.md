# A retrospective analysis on Sales during Covid

This is an analysis project done on sales data during pre-covid and covid years (2019-2023). This retrospective analysis aims to find out about the trends and answer important business questions so that the stakeholders can learn and be better prepared for a similar macro/world wide event were to occur in the future.

The dataset used in this project is based on real datasets. The name of the store has been changed and the data inside were pre-selected and slightly manipulated to maintain anonymity and security of the business. However, the data are as close as the real world business and scenarios likely to encounter for a small scale business.

# Introduction to the Dataset

The dataset is from a US based game store named "GameTown" (hypothetical name). Since it is a US based business, the currencies in the dataset are in US Dollars.

The raw/uncleaned Dataset includes the following columns.  
| Column Name | Description |
|-------------|-------------|
| USER_ID | User identifier |
| ORDER_ID | Order identifier |
| PURCHASE_TS | Purchase timestamp |
| SHIP_TS | Shipping timestamp |
| REFUND_TS | Refund timestamp |
| PRODUCT_NAME | Name of the product |
| PRODUCT_ID | Product identifier |
| USD_PRICE | Price in US Dollars |
| PURCHASE_PLATFORM | Platform where purchase was made |
| MARKETING_CHANNEL | Marketing channel that the purchase was done through |
| ACCOUNT_CREATION_METHOD | Account creation method |
| COUNTRY_CODE | Country code |
